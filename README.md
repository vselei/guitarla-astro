### Pré-requisitos

Ter uma REST API que atenda as requisições. Verifique as  instruções no repo: <a href="https://github.com/vselei/guitarla-strapi">GuitarLA Strapi</a>

##

### Inicialização

Para realizar a instalação dos pacotes:
```
npm install
```

Rodar aplicação:
```
npm run dev
```
